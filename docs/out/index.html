JSCalc

JSCalc is a very simple javascript calculator.

Addition

You can add two numbers by using the JSCalc.add() function.

var result = JSCalc.add(4,3);
result.should.equal(7);

Substraction

You can substract two numbers by using the JSCalc.substract() function.

var result = JSCalc.substract(4,3);
result.should.equal(1);
